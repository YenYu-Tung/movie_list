## environment setup
* install node.js & npm
* clone the repository to local
* $ npm install
* $ npm run start
* see "Express is listening on localhost: 3000." means operate successfully
* ctrl + c to stop server
## development tools
* Node.js 16.5.0
* Express 4.18.1
* Express-Handlebars 3.0.0
* Bootstrap 5.1.3
